Introdução á Tag < img >

A tag img em HTML é usado para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja, não possui uma tag de fechamento e contém apenas atributos que definem o conteúdo visual.

Atibutos Principais:

  • src (source): Define o caminho da imagem que será exibida. Pode ser uma URL externa ou um caminho relativo dentro do servidor.

    Exemplo 1:

    <./imagem1.jpg" style="width: 100%;>

    Resultado:

  • alt (alternative text): fornece um texto alternativo para ser exibido caso a imagem não possa ser carregada. ELe também melhora a acessibilidade para leitores de tela.

    CÓDIGO VAI AQUI
