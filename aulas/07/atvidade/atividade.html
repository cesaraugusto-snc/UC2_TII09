Síntese das Propriedades do CSS

1. color

Define a cor do texto de um elemento.

h1 {
    color:blue;
}

2. text-aling

Alinha o texto dentro de um elemento.

  • left: alinha o texto a esquerda (padrão).
  • center: centraliza o texto.
  • right: alinha o texto a direita.
h1 {
   text-align: center;
}

3. font-size

Define o tamanho de um elemento

h1 {
font-size: 18px;
}

4. background-color

Define a cor de um elemento.

button {
    background-color: yellow;
}

5. color (para botões)

Define a cor para um elemento (usado em conjunto com o background-color).

button {
    color: black;
}
