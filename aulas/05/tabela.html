Introdução as Tabelas em HTML

Tabelas são as estruturas em HTML que permitem a orgainzação de dados em linhas e colunas. São comumente usadas para exibir calendários, horários, lista de preços, etc.

Estrutura Básica de uma Tabela

A estrutura básica de uma tabela em HTML é composta por várias tags:

  • table: Define o inicio e fim de uma tabela.
  • tr: Define uma linha na tabela
  • td: Define uma célula dentro de uma linha (dados)
  • th: Define uma célula de cabeçalho

Exemplo 1: Tabela Básica

Nome Idade Profissão
Ana 28 Engenheira
Bruno 34 Médico

Explicação

  • Nesse exemplo, usamos th para o cabeçalho e td para os dados.
  • O atributo border="1" é usado para adicionar bordas á tabela.
DISCIPLINA UN1 UN2 UN3 UN4

Matemática 8 5 5 5
Inglês 7 8 8 7

Mesclagem de Células

A mesclagem de células pode ser feita tanto em colunas quanto em linhas.

  • colspan: Mescla células horizontalmente (colunas)
  • rowspan: Mescla células verticalmente (linhas)
Nome Idade Profissão
Carlos 30 Professor
31 Arquiteto
Daniela Designer Gráfico

Explicação:

  • rowspan="2" A Célula "Carlos" se expande verticalmente para cobrir duas linhas
  • colspan="2" A Céula "Designer Gráfico" se expande horizontalmente para cobrir as duas colunas.

Adicionando um Cabeçalho e Rodapé a tabela

Além das linhas de dados, uma tabela pode ter um cabeçalho (thead), um corpo (tbody) e um radapé (tfoot)

Produto Quantidade Preço Unitário
Caneta 10 R$ 2,00
Caderno 5 R$ 15,00
Total: R$ 95,00

Explicação:

  • thead: Define o cabeçalho da tabela
  • tbody: Define o corpo da tabela, onde fica os dados
  • tfoot: Define o rodapé da tabela, ideal para somatórios ou resumos
